# Setup Guide

1. Clone this repo

```
git clone https://github.com/remake/remake-framework-new-syntax.git
```

2. Install dependencies

```
cd remake-framework-new-syntax
npm install
```

3. Create a `.remake` file

inside `.remake`:
```
{
  "port": 3000,
  "sessionSecret": "__replace_with_a_random_secret_string__"
}
```

4. Start the project

```
npm run dev
```
